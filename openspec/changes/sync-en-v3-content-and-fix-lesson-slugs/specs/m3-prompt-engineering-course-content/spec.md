### Requirement: M3 prompt-engineering lessons SHALL be fully authored in Chinese MDX
The system SHALL include complete v3-style M3 lesson content for both Chinese and English locales, covering context design, prompt elements, system prompts, and reusable prompt assets.

#### Scenario: Learner opens an M3 lesson in Chinese
- **WHEN** a learner visits any Chinese M3 lesson route
- **THEN** the corresponding MDX file SHALL contain complete Unit-style content with practical examples and exercises

#### Scenario: Learner opens an M3 lesson in English
- **WHEN** a learner visits any English M3 lesson route
- **THEN** the corresponding MDX file SHALL contain English content aligned to the same v3 lesson slugs and pedagogical structure
