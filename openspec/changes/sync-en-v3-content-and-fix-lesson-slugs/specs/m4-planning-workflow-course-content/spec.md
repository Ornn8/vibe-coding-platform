## MODIFIED Requirements

### Requirement: M4 planning-workflow lessons SHALL be fully authored in Chinese MDX
The system SHALL include complete v3-style M4 lesson content for both Chinese and English locales, delivered as five exposed lessons that connect planning artifacts, execution loops, Claude Code onboarding, and a workflow-oriented module project.

#### Scenario: Learner opens an M4 lesson in Chinese
- **WHEN** a learner visits any Chinese M4 lesson route
- **THEN** the corresponding MDX file SHALL contain complete workflow-oriented content aligned to the v3 lesson structure

#### Scenario: Learner opens an M4 lesson in English
- **WHEN** a learner visits any English M4 lesson route
- **THEN** the corresponding MDX file SHALL contain English content aligned to the same five exposed v3 lesson slugs
