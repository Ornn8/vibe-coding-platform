## ADDED Requirements

### Requirement: M4 planning-workflow lessons SHALL be fully authored in Chinese MDX
The system SHALL include complete Chinese MDX lesson content for the M4 learning arc, covering why planning matters, PRD writing, implementation plans, PEV loops, Ask/Plan workflows, and Claude Code onboarding.

#### Scenario: Learner opens an M4 lesson
- **WHEN** a learner visits any Chinese M4 lesson route
- **THEN** the corresponding MDX file SHALL contain complete instructional content and not just topic stubs

#### Scenario: M4 lessons stay workflow-oriented
- **WHEN** an M4 lesson is reviewed
- **THEN** it SHALL present a repeatable workflow, artifact, or loop that the learner can use during AI-assisted building

### Requirement: M4 lessons SHALL bridge from concept learning into execution discipline
Each Chinese M4 lesson SHALL make planning behavior operational by showing how to turn ideas into artifacts, how to verify AI outputs, and how to use terminal-based coding agents responsibly.

#### Scenario: PRD and implementation-plan lessons connect
- **WHEN** the PRD and implementation-plan lessons are reviewed together
- **THEN** they SHALL clearly distinguish "what to build" from "how to sequence the work"

#### Scenario: Claude Code onboarding is actionable
- **WHEN** the Claude Code lesson is reviewed
- **THEN** it SHALL include practical setup, usage guidance, and an exercise that asks the learner to complete a small agent-assisted task
